How Algo Indicators Work

📊 The 5 Indicators Used in This App
RSI
Relative Strength Index (0–100)Measures how fast a stock has been moving. Below 30 = oversold (price may bounce UP — buy signal). Above 70 = overbought (price may fall — sell signal). Best for identifying reversal points.
MACD
Moving Average Convergence DivergenceCompares a 12-day fast average to a 26-day slow average. When the MACD line crosses ABOVE the signal line → BUY. When it crosses BELOW → SELL. Best for trend confirmation.
BB
Bollinger BandsA band drawn 2 standard deviations above and below a 20-day moving average. Price touching the LOWER band = potential BUY. Touching the UPPER band = potential SELL. Squeezing bands = big move coming.
EMA
Exponential Moving Average (9 & 21 day)Faster-reacting average. When short EMA (9) crosses above long EMA (21) = Golden Cross → BUY. When short dips below long = Death Cross → SELL. Used as trend direction filter.
VOL
Volume SpikeA price move with HIGH volume is more reliable. Low volume moves can be fakeouts. This app flags stocks where today's volume is 150%+ of the 10-day average — meaning smart money is likely moving.
🧠 How the Signal Score is Calculated
SCORE
Composite Signal ScoringEach indicator gives a score: +1 for bullish, -1 for bearish, 0 for neutral. RSI, MACD crossover, BB position, EMA crossover, and volume spike are all scored and summed. Score ≥3 = STRONG BUY · Score 1-2 = BUY · Score 0 = HOLD · Score -1 to -2 = SELL · Score ≤-3 = STRONG SELL.

Build this in Python using yfinance + pandas-ta import yfinance as yf import pandas_ta as ta df = yf.download("RELIANCE.NS", period="3mo") df['RSI'] = ta.rsi(df['Close'], length=14) df['MACD'] = ta.macd(df['Close'])['MACD_12_26_9'] df['Signal'] = ta.macd(df['Close'])['MACDs_12_26_9'] df['BB_upper'] = ta.bbands(df['Close'])['BBU_5_2.0'] df['BB_lower'] = ta.bbands(df['Close'])['BBL_5_2.0'] df['EMA9'] = ta.ema(df['Close'], length=9) df['EMA21'] = ta.ema(df['Close'], length=21) print(df.tail())
